# 수사연 — 수학 사고력 연구소

초등학교 수학 수업을 위한 **AI 기반 문제 탐구·변형·평가 플랫폼**입니다.  
학생은 교과서 문제를 스스로 변형하고, 교사는 AI 보조 검토와 시험지 관리로 수업을 운영합니다.

---

## 소개

**수사연(수학 사고력 연구소)**은 초등 수학 교육과정(현재 **초등 4학년 1학기** 단원 중심)에 맞춘 웹 애플리케이션입니다.

| 역할 | 한 줄 요약 |
|------|-----------|
| **학생(연구원)** | 문제를 변형·만들고, 동료 평가·탐구점수·뱃지로 성장 |
| **교사** | 학급 관리, AI 검토 인박스, 시험지·오답노트·학급 문제은행 운영 |

학생 화면에는 **수달 캐릭터(탐구달 → 창의달)** 가 함께하며, 6가지 **변형 전략**을 익히며 수학적 사고력을 키우는 것을 목표로 합니다.

---

## 주요 기능

### 학생

- **기존 문제 변형하기** — 6가지 전략(숫자·상황·조건·구하는 것·조건 추가·역방향)으로 문제 재구성
- **문제 만들기 / 수정하기** — 새 문제 설계 및 편집
- **내 문제 은행 · 학급 문제 풀기** — 개인·학급 문제 저장 및 풀이
- **단원평가 오답노트** — 틀린 문항 재풀이·AI 피드백
- **수학 미니게임** — 스피드 퀴즈, 두더지 잡기, 10 만들기 등
- **탐구 보상** — 탐구점수, 수사관 뱃지, 학급 랭킹, 수달 진화

### 교사

- **학급·학생 관리** — 명단 일괄 등록, PIN 초기화, 학급 코드 발급
- **AI 검토 인박스** — 학생 변형·오답 재풀이 제출물 1차 AI 검토 + 교사 최종 승인
- **시험지 워크플로** — PDF 업로드, 문항 영역 지정, OCR 추출, 시험지 은행
- **스캔 정리** — 학생 답안 스캔 PDF 정렬·채점 보조
- **학급 문제은행** — 승인된 학생 문제를 학급 공유 문제로 관리
- **단원평가 복습** — 교사용 단원평가 검토 화면

### AI 보조 (수사봇)

- 학생 변형 문제의 **정답·전략·윤리(유사도·복붙)** 자동 검토
- 교사 가이드(변형 의도 설명) **적합/부족** 평가
- PDF·이미지 기반 **문항 OCR** (Gemini Vision + Claude 백업)
- 학생 실명·PIN 등 **개인정보는 AI API로 전송하지 않음**
